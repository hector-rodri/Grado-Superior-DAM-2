# README

This project contains a Java program that allows copying files from a source path to a destination path 
The program distinguishes between **text files** and **binary files** and uses different methods for each type:

- **Text files:** copied character by character
- **Binary files:** copied using byte streams  

## Development environment information

- **Operating System:** Windows 10
- **IDE used:** Visual Studio Code
- **Java version:** Java 24  
- **Libraries:** Standard Java libraries: java.io, java.util 

## How to Run the Program

- **Step 1:** Execute Main.java
- **Step 2:** Write the origin and destination route
- **Step 3:** Verify the file copy
