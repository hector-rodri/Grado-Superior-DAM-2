# README

This project contains a Java program that reads and displays data from an **XML** file containing information about football players.  
The program uses **JAXB (Java Architecture for XML Binding)** to automatically convert XML elements into Java objects and show them on the screen

## Development environment information

- **Operating System:** Windows 10  
- **IDE used:** Visual Studio Code  
- **Java version:** Java 24  
- **Libraries:** javax.xml.bind, java.io, java.util

## How to Run the Program

1. Place your players.xml file inside the folder named recursos
2. Run the Main.java file
3. The program will unmarshal (read) the XML file and print all player information in the console
4. Verify the generated files in the project directory
