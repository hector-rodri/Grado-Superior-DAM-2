# README

This project contains a Java program that allows reading and processing an **XML** containing information about football players.  
The program uses the **SAX** technique to parse the XML file, handle its events, and extract specific data for each player efficiently.

## Development environment information

- **Operating System:** Windows 10  
- **IDE used:** Visual Studio Code  
- **Java version:** Java 24  
- **Libraries:** java.io, org.xml.sax, org.xml.sax.helpers  

## How to Run the Program

- **Step 1:** Place your players.xml file in the same folder as Main.java.  
- **Step 2:** Execute Main.java.  
- **Step 3:** Verify the info displayed in the console and the generated files
