# Servicios y Procesos

Este repositorio contiene apuntes, prácticas y material relacionado con la asignatura

## Tema 1: Programación de Procesos (.PROCESOS)

El primer tema introduce la **programación multiproceso** con finalidad de comprender y dominar los fundamentos relacionados con la creación, ejecución, gestión y control de procesos y señales.

La mayoría de los proyectos, ejemplos y ejercicios de esta tema **es preferible compilarlos y ejecutarlos desde la consola Ubuntu**
