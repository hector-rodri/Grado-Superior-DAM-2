PREGUNTA 6. ¿Qué diferencia hay entre el comando cat y less?
El comando cat muestra el contenido completo del fichero de golpe en la terminal. El comando less muestra el contenido de forma paginada, permitiendo avanzar y retroceder

PREGUNTA 7. ¿Para qué sirve la opción -i del comando cp? ¿Cuál es la orden para hacer un alias del comando cp que incluya la opción -i?
La opción -i hace que el comando cp pregunte antes de sobreescribir un fichero que ya exista en el destino. La orden para crear el alias es alias cp='cp -i'

PREGUNTA 8. ¿Qué hace la opción -i del comando rm? ¿Y la opción -i del mv? Escribe la orden para hacer un alias del comando rm que incluya la opción -i.
La opción -i del comando rm hace que pida confirmación antes de borrar cada fichero. La opción -i del mv hace que pida confirmación antes de sobreescribir un fichero destino que ya exista
La orden para el alias es alias rm='rm -i'

PWD: ruta del directorio de trabajo actual
